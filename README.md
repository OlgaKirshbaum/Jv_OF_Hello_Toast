# Jv_OF_Hello_Toast

НАЗВАНИЕ ПРИЛОЖЕНИЯ: Hello Toast

------------------------------
ЧТО ДЕЛАЕТ ПРИЛОЖЕНИЕ:

 Имеет 2 кнопки и счетчик.
 При нажатии 1 - меняется значение счетчика.
 При нажатии 2 - Выводится краткое сообщение (тост).
